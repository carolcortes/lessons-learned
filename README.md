# Projeto Leassons Learned :woman_technologist:
``Abril/2022``

> Projeto desenvolvido no Bloco 3 - Introdução à HTML e CSS, no módulo de Fundamentos de Desenvolvimento Web, na [Trybe](https://www.betrybe.com/).
